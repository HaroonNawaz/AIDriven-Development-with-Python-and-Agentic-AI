# Study Notes Creator - Usage Examples

This document provides practical examples of how to use the study-notes-creator skill effectively.

## Example 1: High School Biology

### Request
```
Create study notes for photosynthesis
Subject: High School Biology
Level: 10th Grade
Source: Chapter 8 of Campbell Biology Textbook
Focus: Light-dependent and light-independent reactions (Calvin Cycle)
```

### Expected Output Structure
- Summary of photosynthesis and its importance
- Key terms: photosynthesis, chloroplast, photon, Calvin Cycle, ATP, NADPH, etc.
- Two main sections: Light-dependent reactions and Calvin Cycle
- Detailed explanations with examples
- Review questions testing comprehension and application
- Easy-to-read formatting with clear headings

## Best Practices for Using This Skill

### 1. Be Specific About Your Needs
- Clearly identify the source material (textbook chapter, lecture, article)
- Specify your academic level
- Mention your learning goals or upcoming assessment

### 2. Provide Context
- Include the course or subject area
- Mention any specific focus areas within the topic
- Indicate if you need theoretical or practical emphasis

### 3. Request Format Variations
- Ask for Cornell notes if you prefer structured note-taking
- Request outline format for hierarchical organization
- Choose summary format for quick reference and revision

### 4. Customize for Your Goals
- Specify if notes are for exam preparation
- Request emphasis on problem-solving if applicable
- Ask for real-world applications if relevant to your field

### 5. Review and Enhance
- Use generated notes as a foundation
- Add personal annotations and examples
- Create flashcards from key terms
- Review and update notes regularly

## Note Structure Overview

All notes generated by this skill follow this consistent structure:

1. **Summary** - Executive overview at the top
2. **Key Terms** - Important vocabulary highlighted and defined
3. **Main Content** - Detailed information organized hierarchically
4. **Review Questions** - Questions for self-assessment at the end
5. **Formatting** - Professional, easy-to-read layout throughout

This structure supports active learning and effective revision preparation.
