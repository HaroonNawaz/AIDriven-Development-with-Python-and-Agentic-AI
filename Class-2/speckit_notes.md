## Markdown Quiz

1.  **Question:** Konsa Markdown syntax ek level 1 heading banata hai?
    **Options:**
    a)  `# Heading`
    b)  `## Heading`
    c)  `* Heading`
    d)  `- Heading`

2.  **Question:** Kis tarah se aap ek word ko **bold** kar sakte hain Markdown mein?
    **Options:**
    a)  `*word*`
    b)  `_word_`
    c)  `**word**`
    d)  `~~word~~`

3.  **Question:** Ordered list banane ke liye konsa syntax use hota hai?
    **Options:**
    a)  `- Item`
    b)  `* Item`
    c)  `1. Item`
    d)  `+ Item`

4.  **Question:** Aap ek hyperlink kaise banate hain jiska text "Google" ho aur URL "https://www.google.com" ho?
    **Options:**
    a)  `[Google](https://www.google.com)`
    b)  `(Google)[https://www.google.com]`
    c)  `{Google}(https://www.google.com)`
    d)  `<Google|https://www.google.com>`

5.  **Question:** Code block banane ke liye konsa syntax use hota hai?
    **Options:**
    a)  `` `code` ``
    b)  ```` ```code``` ````
    c)  `<code>code</code>`
    d)  `// code`

6.  **Question:** Konsa syntax ek horizontal rule (line) banata hai?
    **Options:**
    a)  `---`
    b)  `***`
    c)  `___`
    d)  Upar diye gaye sabhi

7.  **Question:** Aap ek image kaise embed karte hain?
    **Options:**
    a)  `![Alt text](image.jpg)`
    b)  `[Alt text](image.jpg)`
    c)  `<img src="image.jpg">`
    d)  `Image: image.jpg`

8.  **Question:** Ek quote block banane ke liye kya use hota hai?
    **Options:**
    a)  `> This is a quote`
    b)  `<quote>This is a quote</quote>`
    c)  `"This is a quote"`
    d)  `'This is a quote'`

9.  **Question:** Inline code banane ke liye konsa syntax hai?
    **Options:**
    a)  `` `code` ``
    b)  ```` ```code``` ````
    c)  `<code>code</code>`
    d)  `// code`

10. **Question:** Ek italicized word banane ke liye aap kya use karte hain?
    **Options:**
    a)  `*word*`
    b)  `_word_`
    c)  Upar diye gaye sabhi
    d)  `**word**`
